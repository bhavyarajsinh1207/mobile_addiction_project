# Mobile Phone Addiction Risk Prediction Using Logistic Regression

A comprehensive Flask web application that predicts mobile phone addiction risk using machine learning (Logistic Regression). The application provides an interactive interface for users to assess their risk level and offers analytics dashboards with insights from the dataset.

## ✨ Features

### Core Functionality
- **Risk Prediction**: Input user data to predict addiction risk using Logistic Regression
- **Real-time Analysis**: Instant prediction with probability scores
- **Personalized Recommendations**: Custom advice based on risk level

### Dashboard & Analytics
- Interactive charts using Chart.js:
  - Addiction risk distribution
  - Screen time analysis
  - Sleep vs addiction correlation
  - Social media usage patterns
  - Gaming habits comparison
  - Gender-based analysis

### User Interface
- Modern, responsive design with Bootstrap 5
- Mobile-friendly layout
- Smooth animations and transitions
- Clean card-based design with gradient accents

### Error Handling
- Comprehensive form validation
- 404 and 500 error pages
- Model loading error handling
- Data file missing detection

## 🛠 Technology Stack

### Backend
- **Python 3.8+**
- **Flask** - Web framework
- **scikit-learn** - Machine learning
- **pandas** - Data manipulation
- **joblib** - Model persistence

### Frontend
- **Bootstrap 5** - UI framework
- **Chart.js** - Data visualization
- **Font Awesome** - Icons
- **Custom CSS** - Styling enhancements
